Variables y tipos de datos en Python

Python es un lenguaje de tipado dinámico — no necesitas declarar el tipo de una variable, Python lo detecta solo. Es uno de los lenguajes más legibles y fáciles de aprender.

variables.py
# Variables en Python — sin necesidad de const/let
nombre = "Ana"           # str (texto)
edad = 25               # int (entero)
altura = 1.65           # float (decimal)
activo = True            # bool (verdadero/falso)
sin_valor = None         # NoneType (vacío)

# Ver el tipo de dato
print(type(nombre))     # <class 'str'>
print(type(edad))       # <class 'int'>

# f-strings (como template literals en JS)
mensaje = f"Hola, {nombre}. Tienes {edad} años."
print(mensaje)
# Output: Hola, Ana. Tienes 25 años.

# Convertir tipos
numero_texto = "42"
numero_real = int(numero_texto)   # str → int
texto_numero = str(edad)          # int → str

Diferencias clave con JavaScript

comparacion.py
# Python usa indentación (espacios) en vez de llaves {}
# Python usa True/False con mayúscula (JS: true/false)
# Python usa None (JS: null/undefined)
# Python no usa ; al final de las líneas

# Operaciones con strings
nombre = "elizabeth"
print(nombre.upper())      # ELIZABETH
print(nombre.capitalize()) # Elizabeth
print(len(nombre))         # 9
💡 Python vs JavaScript

En Python no usas const ni let. Solo escribes el nombre de la variable y el valor. Python detecta el tipo automáticamente. Las llaves {} se reemplazan con indentación (4 espacios o 1 tab).
