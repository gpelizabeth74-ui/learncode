SELECT — Consultando datos

SQL (Structured Query Language) es el lenguaje para trabajar con bases de datos relacionales. Con SELECT puedes consultar y obtener datos de una tabla.

La tabla con la que trabajaremos

idnombreedadciudadsueldo
1Ana García25Lima3500
2Luis Pérez30Arequipa4200
3María López22Lima2800
4Carlos Ruiz35Cusco5100
5Sofia Ríos28Lima3900
consultas.sql
-- Seleccionar TODOS los datos
SELECT * FROM empleados;

-- Seleccionar columnas específicas
SELECT nombre, ciudad FROM empleados;

-- Filtrar con WHERE
SELECT * FROM empleados
WHERE ciudad = 'Lima';

-- Múltiples condiciones
SELECT nombre, sueldo FROM empleados
WHERE ciudad = 'Lima' AND edad >= 25;

-- Ordenar resultados
SELECT * FROM empleados
ORDER BY sueldo DESC;

-- Limitar resultados
SELECT * FROM empleados
LIMIT 3;
💡 Estructura básica del SELECT

SELECT columnas FROM tabla WHERE condición ORDER BY columna LIMIT número

Solo SELECT y FROM son obligatorios. El resto es opcional según lo que necesites.
