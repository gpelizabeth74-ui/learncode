Variables y tipos de datos en JavaScript

Las variables son contenedores para guardar información. En JavaScript moderno usamos const y let para declararlas.

variables.js
// const — no cambia su valor
const nombre = "Ana";
const edad = 25;
const activo = true;

// let — puede cambiar su valor
let contador = 0;
contador = 1; // ✓ válido

// Los 5 tipos de datos básicos:
const texto = "Hola mundo";    // String
const numero = 42;              // Number
const decimal = 3.14;           // Number
const booleano = false;         // Boolean
const vacio = null;             // Null
let sinDefinir;                  // Undefined

// Ver el tipo con typeof
console.log(typeof texto);    // "string"
console.log(typeof numero);   // "number"
console.log(typeof booleano); // "boolean"

Template literals — strings modernos

template-literals.js
const nombre = "María";
const edad = 22;

// Forma antigua (concatenación):
console.log("Hola, " + nombre + ". Tienes " + edad + " años");

// Forma moderna (template literal con backticks ` `):
console.log(`Hola, ${nombre}. Tienes ${edad} años`);
// Output: Hola, María. Tienes 22 años
💡 Regla práctica

Usa const por defecto. Solo usa let cuando sepas que el valor va a cambiar (contadores, acumuladores). Evita var — es antiguo y tiene comportamientos raros.
