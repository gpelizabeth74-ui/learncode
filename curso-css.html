Selectores CSS

Los selectores son la forma en que CSS apunta a elementos HTML para aplicarles estilos. Es como decirle al navegador: "aplica este estilo a ESTOS elementos".

selectores.css
/* Selector de etiqueta — todos los h1 */
h1 { color: blue; font-size: 32px; }

/* Selector de clase — elements con class="card" */
.card { background: white; padding: 20px; }

/* Selector de ID — el elemento con id="header" */
#header { background: #333; color: white; }

/* Varios elementos a la vez */
h1, h2, h3 { font-family: Arial; }

/* p que esté DENTRO de .card */
.card p { color: #666; }

/* Efecto hover */
.btn:hover { background: darkblue; }

¿Cuándo usar cada uno?

  • Etiqueta (h1, p) → cuando quieres afectar todos los elementos de ese tipo
  • Clase (.nombre) → el más usado, reutilizable en múltiples elementos
  • ID (#nombre) → para un único elemento especial en la página
  • :hover → para efectos al pasar el mouse
💡 Regla de oro

Usa clases casi siempre. Los IDs tienen mayor especificidad y solo puedes usarlos una vez por página.
